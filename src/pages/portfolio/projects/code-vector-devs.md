Bajo la marca comercial **Code Vector**, lideré la arquitectura, el desarrollo y el despliegue de múltiples soluciones de software a medida para el sector privado, consolidando mi experiencia como **Lead Developer**.

## 🚀 Proyectos Destacados

### Farma Clinic
Desarrollo integral de un sistema de gestión para clínicas y farmacias, enfocado en el control de inventario, expedientes de pacientes y optimización del flujo de ventas. Diseñado para alta concurrencia y seguridad de datos médicos.

### Planifica
Plataforma orientada a la gestión operativa. Solución full-stack que permitió a los usuarios digitalizar sus procesos en papel y centralizar su toma de decisiones mediante paneles de control interactivos.

## 💡 Tecnologías utilizadas

- Ecosistema JavaScript / TypeScript
- Frameworks de frontend modernos (React, Angular)
- Backend y APIs RESTful (Node.js, Express)
- Bases de datos relacionales y no relacionales

## 🎯 Enfoque

El objetivo principal como consultor siempre fue entender el modelo de negocio del cliente para entregar herramientas que no solo funcionaran a nivel de código, sino que tuvieran un **retorno de inversión real** en su eficiencia diaria.
